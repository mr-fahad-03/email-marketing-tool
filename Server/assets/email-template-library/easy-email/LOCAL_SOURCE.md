# Easy-Email-Pro Email Templates

Source: https://github.com/Easy-Email-Pro/email-templates

This folder contains a local copy of the Easy-Email-Pro email template collection so the app can list, preview, and edit templates without fetching the template files from GitHub at runtime.

The upstream GitHub README describes the project as open-sourced under the MIT License. The upstream `package.json` currently lists `ISC`, so confirm license terms before redistribution outside this application.
